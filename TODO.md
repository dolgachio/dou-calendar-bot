# План розробки бота

## 1. Бот для спілкування з користувачами

1. Створити тестового бота і використати його для тестування функціоналу.
1. Додати використання реальної D1 бази даних замість `InMemoryUserStore` для зберігання інформації про користувачів.
    - створити D1 базу даних за допомогою Wrangler CLI
    - створити worker для роботи з базою даних Wrangler CLI
    - створити таблицю для зберігання інформації про користувачів 
2. Додати секції з описом того, як він працює.

## 2. Lua скрипт для розсилки повідомлень

1. Створити таблицю для зберігання інформації про вже відправлені новини користувачам. 
2. Доступ до бази даних з Lua скрипта для отримання списку користувачів, які підписані на розсилку.
3. Вичитати RSS стрічку і пропарсити її. 
4. Створити механізм ідентифікації нових подій, які ще не були відправлені користувачам.
